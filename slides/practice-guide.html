練習手冊

自由潛水 Mouthfill 練習指引

1

漏氣練習

📋 步驟
  1. 嘴巴輕輕鼓起來（臉頰要有東西），不要很用力
  2. 鼓氣時鼻子要先捏起來
  3. 捏鼻子 → 讓氣從鼻子小縫漏出去
  4. 手不要整個放掉，只露一個小縫
  5. 發出類似氣球洩氣的聲音
✅ 自我檢查
  • 鼓起來時下巴下方（頦下）也會膨起來
  • 漏氣時臉和頦下要「同時消」
  • 感受發力位置：臉、舌頭前後、喉嚨附近
  • 喉嚨附近的感知最重要
⚠️ 常見錯誤
錯誤正確做法
沒捏鼻子就鼓氣鼓氣一定要捏鼻子
鼻子整隻手放開只露小縫，放開會讓軟顎鎖起來
鼓太用力輕輕鬆鬆讓它膨起來
喉頭被夾在高位喉頭應有明顯上下移動
臉消了但頦下沒消兩個要同時消
🎯 進階：三階段氣量
  • 鼓最大 → 整口漏掉
  • 鼓中間 60-70% → 整口漏掉
  • 鼓少少 → 整口漏掉
每下鼓起來後，先確定後面是放鬆的、有空間的，然後直接送上去。
2

微笑平壓

📋 步驟
  1. 嘴巴含約 60% 空氣
  2. 嘴角跟舌頭一起抬升（不是純舌根）
  3. 臉用力推 → 舌頭同時往上抬抵抗
  4. 兩者互相抵抗，訓練到同步
✅ 自我檢查
  • 臉推時空氣不能被擠下去
  • 要感覺到臉與舌頭的「互相抵抗」
  • 舌頭要有「想撐起來」的感覺
⚠️ 常見錯誤
錯誤正確做法
臉推時空氣被擠下去舌頭要有抵抗感
只有舌根在動嘴角與舌頭一起抬升
刻意做但不是本能練到像含水時舌頭自動擋住一樣自然
目標：練到身體本能 — 含一口水用力往後擠，舌頭自己會起來擋，沒人教你身體本能就會做。
3

開軟顎練習

📋 步驟
  1. 嘴巴輕輕鼓起來
  2. 捏住鼻子
  3. 想像要做一個法蘭茲（不用真的推）
  4. 感覺喉嚨上方有「輕輕掉下來」的感覺
  5. 重點是放鬆，不是用力
✅ 自我檢查
  • 正確感覺：喉嚨上方「輕輕掉下來」的放鬆感
  • 如果感覺是「推開」→ 方向錯了
⚠️ 常見錯誤
錯誤正確做法
舌頭後段上頂（鼓起來後還能用鼻子呼吸）放鬆舌根，是舌頭後段封閉不是軟顎鎖住
用鼻球開關練習用「想像法蘭茲」法（鼻球練習時聲門也在開開關關）
越用力越鎖想「放鬆」不是想「打開」
🔧 感覺不到怎麼辦
  1. 確認舌頭後段沒上頂
  2. 不要想「打開」，要想「放鬆」
  3. 急救法：先往鼻腔灌氣再一起鼓下來
  4. 多下水幾次
  5. 從漏氣練習入手
4

Mouthfill 鼓氣

📋 步驟
  1. 用嘴巴吹氣，同時讓臉鼓起來
  2. 用手指把嘴唇堵起來 → 臉自然膨脹
  3. 同時發出「嗯」或「嗚」的聲音
  4. 肚子要用力（往內縮的感覺）
🌊 水中流程
  1. 前段用法蘭茲/微笑平壓下潛
  2. 到鼓氣深度 → 鼓滿
  3. 露一點面鏡、平耳壓
  4. 再下幾米，補滿第二次
通常鼓 2 次就好。女生建議第一次 8 米、第二次 13 米。女生肺活量較小，肺部更早被壓縮到推不出氣的深度，所以需在更淺處鼓氣。
✅ 自我檢查
  • 鼓氣時肚子往內縮 = 正確，往外凸 = 錯誤
  • 水面鼓一口用到 10-15 米 = 正常（14-15 米及格）
  • 鼓完後後側放鬆蓬蓬的 = 正確
⚠️ 常見錯誤
錯誤正確做法
偷 RP用肚子推，聲門不開
鼓完喉嚨縮上去後側要放鬆蓬蓬的
吞氣（15 米鼓完幾下就沒氣）100% 是開了聲門
小量補氣鼓兩次就好，用肚子推
肚子不用力肚子往內縮，否則變成喉嚨緊
5

調息與全呼吸

📋 調息原則
  • 最好的調息就是正常呼吸
  • 不用很久，確認不喘就好
  • 下潛前心率完全不重要
🔬 每個自潛者都該做的實驗
  1. 第一潛：用你自認很好的調息
  2. 第二潛：只確認不喘，正常吸吐幾口就全呼吸下潛
  3. 你會發現兩趟感覺差不多
✅ 全呼吸自我檢查
  • 吸飽後還能講話 → 沒吸飽
  • 噴一口氣才能講話 → 才是真的吸飽
  • 胸椎挺直，垂直+橫向都要擴張
⚠️ 常見錯誤
錯誤正確做法
用 4-7-8 拼命降心率正常呼吸，做不好就是超呼吸
咬呼吸管出口無法控制超大聲超快 = 超呼吸
調息調太久確認不喘就好
全呼吸沒吸飽噴一口氣才能講話 = 吸飽
全呼吸忽略垂直向擴張胸椎挺直
全呼吸前深呼吸不要，會壓抑呼吸欲望
6

空氣轉換時機

📋 三個階段
  1. 臉推
  2. 舌尖
  3. 舌根
關鍵原則：快推不到就換，不是等到真的沒東西可推才換。
⚠️ 常見錯誤
錯誤正確做法
等到完全推不到才換快推不到就換，太晚會 lock 緊繃
全程只用一種推法分階段利用不同肌群
不知道自己是舌尖還是舌根先用張嘴測試判斷
7

胸腔彈性拉伸

🅰️ 吊胃
  • 判斷法 — 吊胃時沒辦法講話，縮小腹還可以
  • 不用吐超乾
  • 身體要挺起來
  • 每天 5-10 次
🅱️ 按橫膈膜
  • 肋骨下緣找軟的地方，方向「斜上」
  • 懶人法：躺著用手機角斜上戳，吐氣時壓進去
  • 一天按幾次都行
🅲 Packing 自由式
  • 50 米一趟，休息一分鐘，10-20 趟
  • 一週兩次
  • 要有潛伴
✅ 自我檢查
  • Packing 量增加 = 彈性進步
  • 胸腔緊繃感深度門檻往下 = 進步
  • 胸口均勻的緊 = 正常；局部緊 = 有問題
⚠️ 常見錯誤
錯誤正確做法
吊胃吐太乾循序漸進，避免橫膈膜抽筋
下潛時刻意吊胃絕對不要！鎖骨凹陷處會被吸進去
縮身體/含胸身體中立就好
Packing 自由式沒潛伴一定要有潛伴
8

身體中立位

定義：不縮不挺，讓身體回歸自然中立位置。
📋 如何找到中立位
  • 用吊胃感受：身體縮起來橫膈膜就掉下去
  • 下巴、頭、脖子放回來
  • 不是挺胸，只是不鎖
✅ 自我檢查
  • 吊胃測試：橫膈膜抬不起來 = 太縮
  • 下潛時胸腔均勻緊 = 正常；局部異常緊 = 可能太縮
⚠️ 常見錯誤
錯誤正確做法
過度縮（含胸）不鎖胸腔，讓橫膈膜自然抬起
過度挺（刻意吊胃）絕對不要，鎖骨凹陷被吸進去超危險
聽教練說「遇水壓要縮胸腔」這是錯的，中立就好
9

舌位判斷

🔍 張嘴測試
嘴巴張很大時還能平壓 → 可能是舌根平壓或 H-Lock
📊 特徵比較
舌根平壓舌尖平壓
喉頭幅度大動的位置前面、幅度小
容易海豚音更多肌肉可分工
喉頭容易夾高位嘴巴含氣時更輕鬆有力
📋 舌根 → 舌尖轉換方法
  1. 先練舌尖平壓
  2. 再練嘴巴放一點空氣（約60%）的平壓
  3. 含水推擠練習
  4. 目標：連結到微笑平壓
⚠️ 常見錯誤
錯誤正確做法
海豚音（只有舌根在動）多利用前面舌頭
喉頭夾高位不動應有明顯上下移動，做漏氣練習
平壓時脖子爆筋脖子不用力，爆筋容易受傷
不知道自己是哪種就直接練先張嘴測試判斷
